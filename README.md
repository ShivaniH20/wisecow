# 🐄 Wisecow DevOps Project

## 🚀 Deliverables
1. **Docker**: Optimized Dockerfile for the Wisecow app.
2. **Kubernetes**: Manifests for Deployment, Service, and TLS Ingress located in `/k8s`.
3. **CI/CD**: GitHub Actions workflow for automated builds.
4. **Security**: Zero-Trust runtime security using **KubeArmor**.
5. **Automation**: 
   - `scripts/system_health.py`: Monitors CPU/RAM.
   - `scripts/app_health_check.sh`: Verifies app uptime.
